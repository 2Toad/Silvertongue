    • Silvertongue

    Super slick translation transmogrification

    Project maintained by 2Toad Hosted on GitHub Pages — Theme by mattgraham

    Silvertongue was created to ease the burden of maintaining translation files (a.k.a., localization files).

    Plugins

    The modification of translation files (creating/opening/saving) is handled by a plugins. Silvertongue comes with a Plugin API if you're interested in developing a plugin for a product that requires translation files.
